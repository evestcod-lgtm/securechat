# Иконка приложения

Положи сюда свою картинку под именем **icon.png** — квадратная, минимум 1024x1024px, без прозрачности по краям (Android сам обрежет под нужную форму).

Файл должен называться ровно: `resources/icon.png`

После этого иконка применится при следующей сборке APK через GitHub Actions автоматически.
